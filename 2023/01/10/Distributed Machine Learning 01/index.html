Distributed Machine Learning 01

Created2023-01-10|Updated2023-01-21|System

Introduction

Background

主要突出Machine Learning的必要性以及分布式的必要性

  flowchart TD
A(Big Data)--Theory Support-->B(Machine Learning)
C(Distributed)--Performance Support-->B

监督学习（Supervised Machine Learning）下使用大数据进行训练是不可避免的。如果训练集数据有限，假设训练集和测试集是异构的（heterogeneous），通常在test时不如在train时的效果好；假设是同构的（homogeneous），那么比较容易产生过拟合。所以还是得增大数据集去提升train的效果。

于是可以认为数据是ML的基石，为了效果好就要Big Data才行，而Big Data的处理又要分布式来帮助解决。这也自然而然引出了Distributed ML这个领域。

整体上分布式机器学习有以下三个步骤：

  1. 计算：each worker performs calculations over the local model parameter and data, producing some statistics such as gradients;
  2. 聚合：the statistics are aggregated through some communication channel;
  3. 更新：the local model parameters are updated using aggregated statistics

Gradient Optimization

梯度下降是一种最优化手段，有监督的ML问题的目标是找到最小化目标函数 fff 的 θ\thetaθ 值:

arg min⁡θf(θ)=∑i=1NL(xi,yi⏞trainingdata ;θ⏞parameter⏟loss + Ω(θ)⏟regularization(1)arg\space \min\limits_{\theta}{f(\theta)} = \sum^{N}_{i=1} \underbrace{L(\overbrace{x_i,y_i}^{training data}\space; \overbrace{\theta}^{parameter}}_{loss} \space + \space \underbrace{\Omega(\theta)}_{regularization} \tag{1} arg θmin​f(θ)=i=1∑N​lossL(xi​,yi​​trainingdata​ ;θparameter​​ + regularizationΩ(θ)​​(1)

使用梯度下降法更新参数 θ\thetaθ 需要求出梯度向量（gradient vector），要注意梯度是一个向量，是函数变化最快（steepest）的地方:

▽f(θ)≡g(θ)=[∂f∂θ1,∂f∂θ2,⋯∂f∂θs]T(2)\bigtriangledown{f(\theta)}\equiv{g(\theta)} = \lbrack \frac{\partial f}{\partial \theta _{1}} , \frac{\partial f}{\partial \theta _2}, \cdots \frac{\partial f}{\partial \theta _s} \rbrack ^{T} \tag{2} ▽f(θ)≡g(θ)=[∂θ1​∂f​,∂θ2​∂f​,⋯∂θs​∂f​]T(2)

上面公式中的一阶梯度向量是垂直于函数fff的超平面的。同时，在二阶梯度矩阵（Hessian Matrix）中存在s2s^2s2个二阶导数：

▽2f(θ)≡H(θ)≡(∂2f∂θ12⋯∂2f∂θ1∂θs⋯⋯∂2f∂θs∂θ1⋯∂2f∂θs2)(3)\bigtriangledown ^2{f}{(\theta)} \equiv {H(\theta)} \equiv \begin{pmatrix} \frac{\partial^{2} f}{\partial \theta _{1}^{2}} & \cdots & \frac{\partial^{2} f}{\partial \theta _{1} \partial \theta_{s}} \\ \cdots & & \cdots \\ \frac{\partial^{2} f}{\partial \theta _{s} \partial \theta_{1}} & \cdots & \frac{\partial^{2} f}{\partial \theta _{s}^{2}} \\ \end{pmatrix} \tag{3} ▽2f(θ)≡H(θ)≡⎝⎜⎛​∂θ12​∂2f​⋯∂θs​∂θ1​∂2f​​⋯⋯​∂θ1​∂θs​∂2f​⋯∂θs2​∂2f​​⎠⎟⎞​(3)

超平面
超平面是一个N维空间中的N-1维几何对象，它由一组N-1维的向量和位移量构成。比如，在3维空间中，超平面就是一个2维的平面，由三个向量和位移量构成。

求得梯度目的是为了更新参数，以一阶梯度为例：

θt+1=θt−ηf′(θt)(4)\theta_{t+1} = \theta_{t} - \eta f^{'}{(\theta_{t})} \tag{4} θt+1​=θt​−ηf′(θt​)(4)

基于梯度下降的优化方法有四种：

Method Detail
Steepest descent method 使用一阶梯度更新参数
Conjugate gradient algorithm 使用一阶梯度同时考虑历史梯度
Newton method 使用二阶梯度更新参数
Quasi-Newton method 使用一阶梯度近似得到Hessian Matrix

Steepest Descent Method

最速下降法是最简单但也是最常用的，尤其是其开销比较小，其他的方法在计算、内存占用和通信方面效率低下，所以分布式ML中一般关注机器学习中的一阶梯度优化

g(θ)≡f′(θ)≡▽f(θ)(5)g(\theta) \equiv f'(\theta) \equiv \bigtriangledown f{(\theta)}\tag{5} g(θ)≡f′(θ)≡▽f(θ)(5)

Conjugate Gradient Algorithm

共轭梯度法是一种迭代方法，用于求解最优化问题中的最小值，是求解稀疏对称正定线性方程组的最流行和最著名的迭代技术之一。

共轭梯度法通过在当前步骤计算出一个新的搜索方向，与上一步的搜索方向相反，来有效地寻找最优值。可以用来求解凸优化问题和非凸优化问题。

计算第t轮的共轭梯度为：

pt+1=αtpt−g(θt)(6-1)p_{t+1} = \alpha_{t}p_{t}-g(\theta_{t}) \tag{6-1}pt+1​=αt​pt​−g(θt​)(6-1)

α=g(θk)Tg(θk)g(θk−1)Tg(θk−1)(6-2)\alpha = \frac{g(\theta_k)^{T}g(\theta_k)}{g(\theta_{k-1})^{T}g(\theta_{k-1})} \tag{6-2} α=g(θk−1​)Tg(θk−1​)g(θk​)Tg(θk​)​(6-2)

Newton Method

从几何上说，牛顿法就是用一个二次曲面去拟合你当前所处位置的局部曲面，而梯度下降法是用一个平面去拟合当前的局部曲面，通常情况下，二次曲面的拟合会比平面更好，所以牛顿法选择的下降路径会更符合真实的最优下降路径。

可以用泰勒展开到二阶来解释牛顿法：

f(θ+δ)≈f(θ)+g(θ)Tδ+12δTH(θ)δ(7)f(\theta + \delta) \approx f(\theta) + g(\theta)^{T}\delta + \frac{1}{2} \delta^{T}H(\theta)\delta \tag{7} f(θ+δ)≈f(θ)+g(θ)Tδ+21​δTH(θ)δ(7)

其中H(θ)H(\theta)H(θ)为Hessian矩阵，函数增量近似为：

f(θ+δ)−f(θ)=g(θ)Tδ+12δTH(θ)δ(8)f(\theta + \delta) - f(\theta) = g(\theta)^{T}\delta + \frac{1}{2} \delta^{T}H(\theta)\delta \tag{8} f(θ+δ)−f(θ)=g(θ)Tδ+21​δTH(θ)δ(8)

求导解得增量最小时：

δ=−H(θ)−1g(θ)(9)\delta = -H(\theta)^{-1}g(\theta) \tag{9} δ=−H(θ)−1g(θ)(9)

总的来说，牛顿法优缺点如下：

  • 优点：二阶收敛，收敛速度快

  • 缺点：牛顿法每一步都要求目标函数的Hessian矩阵的逆矩阵，计算比较复杂

Quasi-Newton Method

拟牛顿法是在一阶梯度下降和二阶梯度下降之间的一个trade-off，拟牛顿方法提出使用先前的一阶梯度来近似Hessian Matrix，一定程度上降低了计算量

First Order Gradient Optimization Algorithm

主要有三种常见手段：批量梯度下降、随机梯度下降（SGD）、小批量梯度下降

Batch Gradient Descent

是一种比较原始的想法，它使用整个训练数据并计算梯度向量，但是显然这样比较低效，尤其是大数据集时。甚至在数据是流式产生的情景下，其根本不可用。

Stochastic Gradient Descent

随机梯度下降（SGD）从训练数据中随机选取一个实例，并在该实例上计算梯度向量用于更新参数，这样产生的随机梯度之间具有较高的方差，在许多情况下产生不稳定的收敛。但理论上，如果学习率随着训练的进行而衰减，SGD也能保证在凸优化问题上收敛到全局最优。

Minibatch Gradient Descent

批量梯度下降导致计算缓慢，而随机梯度下降遭受较大的梯度方差，这两种方法之间的trade-off是在每次迭代时使用最小的一批训练实例。

这种办法效率和效果都不错，同时根据其思想产生的一些变体可以来解决学习率的选择和数据稀疏之类的问题，比如Momentum、Adagrad、Adam

Momentum

如果我们把梯度下降法当作小球从山坡到山谷的一个过程，那么在小球滚动时是带有一定的初速度，在下落过程，小球积累的动能越来越大，小球的速度也会越滚越大，更快的奔向谷底，受此启发就有了动量法 Momentum。

动量的作用原理：利用惯性，即当前梯度与上次梯度进行加权，如果方向一致，则累加导致更新步长变大；如果方向不同，则相互抵消中和导致更新趋向平衡。动量 γ 常被设定为0.9或者一个相近的值：

mt+1=γmt+ηg(θt)(10-1)m_{t+1} = \gamma m_{t} + \eta g(\theta_{t}) \tag{10-1} mt+1​=γmt​+ηg(θt​)(10-1)

θt+1=θt−mt(10-2)\theta _{t+1} = \theta _{t} - m_{t} \tag{10-2} θt+1​=θt​−mt​(10-2)

其中η\etaη是学习率，g(θt)g(\theta_{t})g(θt​)是梯度

总的来说，Momentum方法的目的是减少学习速率和最小化损失函数。它通过将一部分前一步的梯度（Momentum）和当前步的梯度相加来使梯度下降趋于平稳，从而减少参数在局部最优解附近“跳跃”的可能性。

Adagrad

能否根据参数的重要性自适应学习率呢？Adagrad 的提出思想是：在学习的过程中自动调整学习率。对于出现频率低的参数使用较大的学习率，出现频率高的参数使用较小的学习率。Adagrad为每一个参数 θi\theta{i}θi 提供了一个与自身相关的学习率，公式如下:

θt+1,i=θt,i−ηGt,i+εgt,i,  Gt,i=∑j=0t∣∣gj,i∣∣2(11)\theta_{t+1,i} = \theta_{t, i} - \frac{\eta}{\sqrt{G_{t,i} + \varepsilon}}g_{t,i}, \space \space G_{t,i} = \sum^{t}_{j=0} ||g_{j,i}||^{2} \tag{11} θt+1,i​=θt,i​−Gt,i​+ε​η​gt,i​,  Gt,i​=j=0∑t​∣∣gj,i​∣∣2(11)

Adagrad是为了解决参数列表稀疏性异质性问题而提出的。Adagrad没有为每个梯度维度使用相同的学习率，而是为每个维度选择单独的学习率。简言之，Adgrad记录所有梯度维度的过去变化，并动态调整每个维度的学习率，使其与过去的变化成反比。

Adam

自适应矩估计算法Adam（Adaptive Moment Estimation）结合了Momentum和Adagrad技巧的一阶梯度优化算法。其包含了动量项mmm（累加过去的梯度）以及一个过去梯度的平方项：

mt+1=β1mt+(1−β1)g(θt)(12-1)m_{t+1} = \beta_{1}m_{t} + (1-\beta_{1})g(\theta_{t}) \tag{12-1} mt+1​=β1​mt​+(1−β1​)g(θt​)(12-1)

vt+1=β2vt+(1−β2)g2(θt)(12-2)v_{t+1} = \beta_{2}v_{t} + (1-\beta_{2})g^{2}(\theta_{t}) \tag{12-2} vt+1​=β2​vt​+(1−β2​)g2(θt​)(12-2)

其中mtm_tmt​是以β1\beta_1β1​为衰减因子的历史梯度的流动平均值，vtv_tvt​是以β2\beta_2β2​为衰减因子的历史梯度的流动平均值。一般把β1\beta_1β1​和β2\beta_2β2​设置为接近1的值，比如分别为0.9、0.999但这样会导致训练开始时产生冷启动的问题，所以一般会进行偏差矫正：

m^t=mt1−β1t(13-1)\hat{m}_{t} = \frac{m_t}{1-\beta^{t}_{1}} \tag{13-1} m^t​=1−β1t​mt​​(13-1)

v^t=vt1−β2t(13-2)\hat{v}_{t} = \frac{v_t}{1-\beta^{t}_{2}} \tag{13-2} v^t​=1−β2t​vt​​(13-2)

最终，参数更新方式为：

θt+1=θt−ηvt^+εm^t(14)\theta_{t+1} = \theta_{t} - \frac{\eta}{\sqrt{\hat{v_t}}+\varepsilon}\hat{m}_{t} \tag{14} θt+1​=θt​−vt​^​​+εη​m^t​(14)

其中ε\varepsilonε一般取10−810^{-8}10−8

Adam由于其快速性能和无超参数机制而成为最流行的一阶梯度优化算法
